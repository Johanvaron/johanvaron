## 🚀 Sobre mí
- 🧠 Construyo **aplicaciones con IA**: agentes conversacionales y **bots de voz en tiempo real** (STT → LLM → TTS).
- 🏗️ Desarrollo **plataformas SaaS multi-tenant** de punta a punta: API, dashboard, workers, infra y despliegue.
- ☁️ **Self-hosting / DevOps**: todo dockerizado en mi propio VPS Linux, con Postgres, Redis, colas y reverse proxy.
- 🤝 Integro **WhatsApp, Shopify y Meta Ads** para automatizar marketing y e-commerce.
